Este laboratório faz parte do programa AWS re/Start V3, no qual estou participando.
Meu primeiro laboratório com Amazon EC2, onde aprendi a iniciar e configurar uma instância na nuvem. 
O exercício incluiu habilitar proteção contra encerramento, configurar grupo de segurança, 
instalar um servidor web simples com Apache e monitorar métricas básicas via CloudWatch.
Foi uma ótima introdução prática ao uso de instâncias EC2 e à gestão de recursos na AWS.
